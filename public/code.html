Code Tutorial

The code tag <code> is used to display text as it is displayed when writing code. In simpler terms, it displays a piece of computer code as is. It requires both an opening tag <code> and a closing tag </code>. In between those two tags are the inserted text.

<code> Text <code>

This is an example of the code:

<code> TEXT <code>

This is an example of how the web page displays the code:
