Paragraph Tutorial

The paragraph <p> tag is used to insert a paragraph or text onto a web page. It requires an opening tag <p> and a closing tag </p>. The text inserted in between the paragraph tags is what appears on the web page.

Fun Fact: To create a return within lines of text, you need to create two different paragraph tags. Such as:

<p> I went for a walk. </p>

<p> It was fun. </p>

I went for a walk.

It was fun.

This is the code:

<p> This is a paragraph </p>

<p> This is yet another paragraph </p>

This is how the web page displays the code:

This is a paragraph
