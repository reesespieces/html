Button Tutorial

The button tag <button> is used to insert buttons onto a web page. The button tag requires an opening tag <button> and wihtin that opening tag you insert the type. For example, <button type="button">. The button tag also requires a closing tag </button>. In between the opening tag and the closing tag, the user inputs the text or images that they want to appear on the button. Such that:

<button type="button"> Text </button>

The button tag has multiple attributes that can allow the user to call on a link or text when the button is pressed. These attributes are inserted in the opening tag after the type="button" attribute. The attribute for calling upon text is onclick. For example, to have window appear when you press a button, you would write:

<button type="button" onclick="alert('Hello World!')"> Click Me </button>
