Image Tutorial

The image tag <img> is used to insert an image onto a web page. The image tag is different than tags like the paragraph tag (<p> and </p>) because within the <img>, you insert the image and image preferences (such as height and width). So this is what the code would be structured like:

<img src = "/image.jpg" height="x amount of pixels as a numerical value" width="x amount of pixels as a numerical value">

This is an example of what the code looks like:

<img src="/images/sloth.jpg" height="250" width="300">
