Lists Tutorial

The list element < li > is used to insert a list. However, there are two different types of lists that can be inserted.

  • Unordered Lists
  1. Ordered Lists

Unorder Lists

Unordered lists <ul> are used to insert lists using generic bullet marks. It requires both an opening tag <ul> and a closing tag </ul>. However, nested in between the opening tag <ul> and the closing tag </ul>, are the list element tags <li> and </li> with text in between these tags. For example:

<ul><li> Text </li></ul>

Ordered Lists

Ordered lists <ol> are used to insert numbered lists. It requires both an opening tag <ol> and a closing tag </ol>. However, nested in between the opening tag <ol> and the closing tag </ol>,are the list element tags <li> and </li> with text in between these tags. For example:

<ol><li> Text </li></ol>

This is what the code looks like:

<ul><li> This is an unordered list </li></ul>

<ol><li> This is an ordered list </li></ol>

This is what the web page displays:

  • This is an unordered list
  1. This is an ordered list
